# SEO-Friendly Image Naming Plan for On-Point Insulation

This document provides a practical plan for renaming On-Point Insulation images with SEO-friendly names that accurately reflect the content while improving search engine visibility.

## Naming Convention Guidelines

Each image filename should follow this structure:
```
onpoint-sprayfoam-[location]-[surface-type]-[phase]-[number].webp
```

Key components to include:
- **Company name**: Always start with "onpoint-"
- **Product specification**: "sprayfoam-" or "2lb-closed-cell-" for specific product mentions
- **Location**: "durango" or "colorado" 
- **Surface/Application**: "wall", "ceiling", "construction", "framing", "new-construction"
- **Phase**: "before", "during", "after", "closeup"
- **Number**: Sequential numbers for similar images

## Implementation Notes

1. Keep the names accurate to what is actually shown in the images
2. Group similar images with the same naming pattern for consistency
3. Use the sequential numbering to distinguish between similar images
4. Consider organizing images in folders by project type for easier management
5. Add descriptive alt text to each image when adding to the website

## Technical Tips

1. When implementing, rename images in small batches to avoid errors
2. Keep your original files as backup until the renaming is complete
3. Update all website references after renaming
4. Consider using automated tools to help with bulk renaming
